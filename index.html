CPU Scheduler Simulator

A command-line Java application that simulates various CPU scheduling algorithms. This project properly implements multithreading to simultaneously demonstrate algorithms such as First-Come-First-Serve, Shortest Job First, Priority Scheduling, and Round Robin. The simulator provides detailed metrics on process execution, waiting times, and throughput, making it a valuable educational tool for understanding operating system concepts.
